## 🧠 From Jupyter Notebook to Modular Coding

I broke everything out into proper Python modules — separate files for data ingestion, transformation, model training, prediction, and logging. Each piece does one job, and does it well.

## 🤖 I Didn't Pick One Algorithm. I Picked Ten.

Because why not.

- Random Forest
- XGBoost
- CatBoost
- AdaBoost
- Lasso Regression
- Ridge Regression
- Support Vector Regressor (SVR)
- Decision Tree
- K-Nearest Neighbors (KNN)
- Linear Regression

All hyperparameter-tuned with `GridSearchCV`.

The pipeline basically interviews all 10 models, puts them through a performance review, and only the best one gets the job.

## 📋 I Added Logging. 

Every step of the pipeline writes logs. Every error tells you the exact file and line number where things went wrong — no more guessing games.

## 🌐 Flask API + HTML Frontend 

Not "live" as in running on my laptop at 2am. Actually deployed. Anyone can use it. Wild concept.
